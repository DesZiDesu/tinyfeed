# TinyFeed

โทรศัพท์จิ๋วที่มีฟีดโซเชียลและข่าวสารในตัวเดียว สำหรับเสริม worldbuilding ในการโรลเพลย์

## Installation
1. เปิด SillyTavern
2. Extensions → Install Extension
3. วาง GitHub URL
4. Refresh หน้าเว็บ

## Testing
มองหาปุ่ม TinyFeed ในเมนู extensions (ไอคอนไม้กายสิทธิ์ข้างกล่อง input)
