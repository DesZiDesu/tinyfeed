# TinyFeed — TinyPhone 📱

โทรศัพท์จำลองสำหรับโรลเพลย์ใน **SillyTavern** — มีหน้าโฮมและแอปในตัว 3 แอป ให้ตัวละคร/NPC มีชีวิตนอกบทสนทนาหลัก และเสริมบรรยากาศ worldbuilding

> เปิดจากเมนู extensions (ไอคอนไม้กายสิทธิ์ข้างกล่องพิมพ์) → เข้าหน้าโฮม → เลือกแอป

---

## 📲 แอปในเครื่อง

### 📱 TinyFeed — ฟีดโซเชียล + ข่าวสาร
- **แท็บฟีด** — โพสต์เอง หรือกดให้ AI สร้างโพสต์ในนามตัวละครหลัก/NPC
- **ไลค์ · แชร์ · คอมเมนต์** พร้อมหน้ารายละเอียดโพสต์
- **AI ตอบคอมเมนต์** 2 โหมด (ตอบทันที / กดปุ่มขอ) — AI เลือกเองว่าจะตอบในนามเจ้าของโพสต์หรือ NPC
- **คอมเมนต์ NPC ติดมากับโพสต์ใหม่** เลือกได้ (ไม่มี / ให้ AI เลือกจำนวน / กำหนดจำนวนเอง)
- **สร้างโพสต์อัตโนมัติ** เมื่อคุยในแชทครบ N ข้อความ — 2 โหมด (ครบแล้วสร้างเลย / ให้ AI ตัดสินใจก่อนว่าควรโพสต์ไหม)
- **แท็บข่าวสาร** — AI แต่งข่าว/บทความเสริมสถานการณ์โลก (กดสร้างเอง หรืออัตโนมัติ ตั้งค่าแยกจากฟีด) อ่านบทความเต็มได้
- ยอดไลค์เริ่มต้นสุ่มตามช่วงที่ตั้ง · เลขเยอะย่อเป็น K/M ในฟีด แสดงเต็มเมื่อกดเข้าโพสต์

### 💬 TinyConnect — แชตสไตล์ไลน์
- **แชตเดี่ยว** กับตัวละครหลักและ NPC ประจำ · AI ตอบในบทบาทนั้น
- **แชตกลุ่ม** — สร้างกลุ่ม ตั้งชื่อ เลือกสมาชิก (≥2 คน) · AI ให้สมาชิก 1–2 คนคุยตอบกัน พร้อมชื่อ+รูปกำกับแต่ละคน · ลบกลุ่มได้
- บับเบิลสไตล์ไลน์ (เรา = ขวา, คู่สนทนา = ซ้าย) · อินดิเคเตอร์ "กำลังพิมพ์…"

### 🎥 TinyStream — ไลฟ์สตรีม + คอมเมนต์สด
- **เริ่ม/จบไลฟ์** — AI ตั้งหัวข้อไลฟ์จากเนื้อเรื่องให้อัตโนมัติ + สุ่มจำนวนผู้ชม
- **สตรีมเมอร์เลือกได้** — ตัวละครหลัก หรือ ตัวเราเอง
- **คอมเมนต์สดจากผู้ชม AI** 3 โหมด — กดปุ่มโหลด / ไหลอัตโนมัติตามเวลา / ไหลตอนเราคอมเมนต์
- เราพิมพ์คอมเมนต์สดเองได้ · มีเส้นคั่น "เริ่มไลฟ์/จบไลฟ์" แยกเซสชันชัดเจน

## 🔗 การเชื่อมกับโรลเพลย์

- **แทรกเข้าประวัติแชทหลัก** — ให้โมเดล RP รับรู้เนื้อหาในโทรศัพท์แบบเรียลไทม์ (ผ่าน `setExtensionPrompt` ตำแหน่ง IN_CHAT กำหนด depth ได้) เลือกได้ว่าจะแทรกโพสต์ / โพสต์+ข่าว / แชต TinyConnect / ไลฟ์ TinyStream → ตัวละครอ้างอิงถึงได้ เช่น *"เห็นที่โพสต์แล้วนะ"*
- **เชื่อมเนื้อหาข้ามแอป** — เปิดแล้วตอน AI สร้างเนื้อหาในแอปหนึ่งจะเห็นเนื้อหาแอปอื่นด้วย (คุยในแชตแล้วพูดถึงโพสต์ในฟีด / ผู้ชมไลฟ์พูดถึงข่าว)

## 🤖 ระบบ Generation

- ใช้ `generateQuietPrompt()` ของ SillyTavern — สร้างเนื้อหาโดย**ไม่ปนกับบทสนทนาหลัก**
- **เลือกโมเดล/API แยกได้** ผ่าน **Connection Manager** ของ ST (เช่นใช้โมเดลถูก/เร็วสำหรับโพสต์ แยกจากโมเดลตอบ RP) — มี fallback กลับ API หลักอัตโนมัติถ้า profile มีปัญหา
- โหมดโมเดลแยกจะแนบบริบทให้เอง แยกหัวข้อชัดเจน: **ตัวละครหลัก** (คำอธิบาย/บุคลิก/ฉาก) · **ผู้ใช้ (Persona)** · **World Info** (ครบทุกตำแหน่งที่ ST inject — ก่อน/หลังคำอธิบายตัวละคร, ก่อน/หลัง Author's Note, แทรกในบทสนทนาตาม depth, ตัวอย่าง, outlet) · **บทสนทนาล่าสุด**
- **ตัดส่วน reasoning/thinking อัตโนมัติ** (อ่าน tag จากการตั้งค่า reasoning ของ ST + tag ยอดนิยม) — โมเดลที่คิดก่อนตอบก็ใช้ได้ ไม่มี `<think>` หลุด
- ตั้ง **จำนวน token** และ **คำสั่งเสริม (prompt)** ได้แยกทุกแอป

## 🎨 หน้าตาและการใช้งาน

- **ธีมสว่าง/มืด** สลับได้ในเครื่อง · **วอลเปเปอร์หน้าโฮม** ใส่ลิงก์รูปได้
- **แจ้งเตือนสไตล์โทรศัพท์** — แบนเนอร์เลื่อนลงจากบนจอ (กดแล้วเข้าแอปที่เกี่ยว) + **จุดแดงนับเลข**บนปุ่มเมนู + ปุ่มสั่น + สั่นบนมือถือ · ปิดได้
- **Markdown เบาๆ** — `**ตัวหนา**` `*เอียง*` `~~ขีดฆ่า~~` `` `โค้ด` `` และ **#แฮชแท็ก / @เมนชัน เป็นสีธีม**
- **ระบบรูปโปรไฟล์** — ตัวละครหลักใช้รูปการ์ด · ผู้ใช้ใช้รูป persona · NPC ใช้รูปที่กำหนด หรือวงกลมสีจากตัวอักษรแรก (ชื่อเดิม = สีเดิมเสมอ) · **override ด้วยลิงก์ภายนอกได้ทั้งผู้ใช้และรายตัวละคร**
- **เวลาแบบสัมพัทธ์** (เมื่อสักครู่ / X นาที / ชั่วโมง / วันที่แล้ว) อิงเวลาจริง
- แอนิเมชันนุ่มนวล · empty state · skeleton ตอนโหลด · รองรับทั้งจอคอมและมือถือ

## 💾 การเก็บข้อมูล

- **ผูกกับแชทแต่ละอัน** (`chat_metadata.tinyfeed`) — ฟีด · ข่าว · NPC ประจำ · แชต/กลุ่ม TinyConnect · สถานะไลฟ์ TinyStream
  → สลับแชท/ตัวละคร ข้อมูลเปลี่ยนตาม **ไม่ปนกัน** และไม่หายเมื่อรีเฟรช
- **ตั้งค่าส่วนกลาง** (`extension_settings.tinyfeed`) — ธีม การแจ้งเตือน โมเดล/API วอลเปเปอร์ และค่า config ของทุกแอป

## 📥 การติดตั้ง

1. เปิด SillyTavern
2. **Extensions → Install Extension** แล้ววาง GitHub URL
3. Refresh หน้าเว็บ (Ctrl+F5)
4. กดไอคอนไม้กายสิทธิ์ข้างกล่องพิมพ์ → เลือก **TinyFeed**

> ⚠️ ชื่อโฟลเดอร์ต้องเป็น `tinyfeed` เท่านั้น (ตรงกับ `extensionName` ในโค้ด)

**ติดตั้งเอง (แบบ manual):** วางไฟล์ทั้งหมดไว้ที่
`SillyTavern/public/scripts/extensions/third-party/tinyfeed/`

### ความต้องการ
- SillyTavern เวอร์ชันที่มี `getContext()` พร้อม `generateQuietPrompt`, `setExtensionPrompt`, `getThumbnailUrl`
- (ไม่บังคับ) **Connection Manager** ถ้าต้องการใช้โมเดล/API แยก

---

## 🚀 เริ่มใช้งานเร็วๆ

1. เปิดแชทที่มีตัวละคร → เปิด TinyFeed → เข้าแอป **TinyFeed**
2. กด **"ให้ตัวละครโพสต์"** ดูผลลัพธ์
3. เปิด ⚙️ → เพิ่ม **NPC ประจำ** สัก 2–3 ตัว (ใช้ได้ทั้งฟีด แชต และไลฟ์)
4. อยากให้ตัวละครรู้เรื่องในโทรศัพท์ → ⚙️ → **แทรกฟีดเข้าประวัติแชท**
5. อยากให้เนื้อหาแต่ละแอปอ้างอิงกัน → เปิด **เชื่อมเนื้อหาข้ามแอป**

> 💡 ฟีเจอร์อัตโนมัติทุกตัวเรียก AI จริง = ใช้ token เพิ่ม ตั้งความถี่ให้เหมาะกับการใช้งาน

```
tinyfeed/

├── index.js         ตรรกะทั้งหมด (แอป, generation, storage, navigation)
├── phone.html       โครง UI โทรศัพท์ + หน้าโฮม + ทุกแอป + หน้าตั้งค่า
├── style.css        ธีม สี แอนิเมชัน (ตัวแปร --tf-*)
├── settings.html    drawer ในแผง Extensions (toggle แสดงปุ่มในเมนู)
└── README.md
```

## 🛠️ หมายเหตุสำหรับการพัฒนาต่อ

- ตัวแปรธีมทั้งหมดเป็น `--tf-*` บน `#tinyfeed-phone` (แยก dark/light)
- คลาส `.tinyfeed-hidden` ใช้ `display: none !important` **โดยตั้งใจ** — เพราะ rule ที่ตั้ง `display` ทีหลังเคยชนะมันจน UI หลุดข้ามหน้า
- เพิ่มแอปใหม่: เพิ่มไอคอนในหน้าโฮม (`.tinyfeed-app-icon[data-app]`) + view `.tinyfeed-app` + เคสใน `openApp()`
- เพิ่มหมวดตั้งค่าใหม่: เพิ่ม `.tinyfeed-settings-group` แล้วใส่ชื่อหัวข้อลง `SETTINGS_LAYOUT` (ถ้าไม่ใส่ จะไปต่อท้ายให้อัตโนมัติ ไม่หาย)
